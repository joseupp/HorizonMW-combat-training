Combat training for HMW-mod.

You can level up against bots.

Uses the DoktorSAS autobot script to add bots (credits to DoktorSAS).

You can choose the map and game mode, and change the game mode options from the menu.

The bot option in the menu does not work.

It is configured with 4xp and 11 bots with mixed difficulty.

Works with the latest version of HMW-mod v1.6.1.

Place the hmw-files.ff file in Program Files (x86)\SteamLibrary\steamapps\common\Call of Duty Modern Warfare Remastered\zone
